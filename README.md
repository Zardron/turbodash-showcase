# Turbo Dash Showcase

This is a public showcase repository for **Turbo Dash**. The production source code is kept private to protect the full implementation, business logic, security-sensitive architecture, and project-specific assets.

## Live Project

https://turbodashdelivery.com/

## What This Project Demonstrates

- Courier service landing pages
- Tracking/contact/registration flows
- Admin notification dashboard
- Maintenance mode controls
- Hostinger deploy workflow

## Tech Stack

- React
- Vite
- TypeScript
- Express
- MongoDB
- Mongoose
- JWT auth
- Socket.IO
- Tailwind CSS

## Public Repository Scope

This repository intentionally contains only a high-level folder outline and safe placeholder files. It is meant for portfolio review, project discovery, and architecture overview only.

The private source repository contains the complete application code, integrations, environment handling, database logic, authentication flows, and deployment configuration.
